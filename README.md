## Development

This project requires Java 8 and Spigot 1.16.5.

Compile it and add it to your local Maven Repository with [BuildTools](https://www.spigotmc.org/wiki/buildtools/).

```bash
java -jar BuildTools.jar --rev 1.16.5
```
